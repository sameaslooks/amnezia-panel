## 📌 О проекте

**Amnezia VPN Panel** — это профессиональная веб-панель для управления VPN-серверами на базе протокола AmneziaWG2. Проект предоставляет полный цикл управления: от установки сервера до мониторинга трафика и лимитов пользователей.

### 🎯 Для кого это?
- **Провайдеры VPN-услуг** — полный контроль над пользователями и серверами
- **Администраторы** — централизованное управление несколькими серверами
- **Разработчики** — открытый код для кастомизации и интеграций

### 🏢 Коммерческое использование
Проект идеально подходит для построения VPN-бизнеса. Для коммерческих целей доступен расширенный **биллинг-модуль** (закрытый), который добавляет:
- Интеграцию с платежными системами
- Автоматическое создание/продление подписок
- Тарифные планы и промокоды
- Email-уведомления
- API для внешних биллинговых систем

*По вопросам приобретения биллинг-модуля обращаться к разработчику*

## ⚡ Быстрый старт

### Системные требования
- **ОС**: Linux (рекомендуется Ubuntu 20.04 / 22.04 / 24.04)
- **Процессор**: 1 vCPU (минимум)
- **RAM**: 1 GB (минимум), 2 GB (рекомендуется)
- **Диск**: 10 GB свободного места
- **Docker**: версия 20.10+
- **Docker Compose**: версия 2.0+

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/sameaslooks/amnezia-panel
cd amnezia-panel

# Настройка окружения
cp .env.example .env
cp docker-compose.yml.example docker-compose.yml
cp nginx.conf.example nginx.conf

# Редактирование конфигурации
nano .env  # Обязательно укажите JWT_SECRET!

#Важно! В файле .env необходимо изменить:

#    JWT_SECRET - установите сложный секретный ключ

#    При необходимости настройте другие параметры

# Запуск панели
docker compose up -d

# Просмотр логов
docker compose logs -f

```

## 🔧 Конфигурация
Переменные окружения (.env)
```env

# Обязательные параметры
JWT_SECRET=ваш-супер-секретный-ключ-измените-это

# Опциональные параметры
DEBUG=false                    # Режим отладки (true/false)
DB_PATH=/app/data/amnezia.db   # Путь к базе данных
LOG_LEVEL=INFO                  # Уровень логирования

# Параметры по умолчанию (можно не менять)
DEFAULT_CONFIG_LIMIT=1          # Лимит конфигов для новых пользователей
TRAFFIC_COLLECT_INTERVAL=60     # Интервал сбора статистики (секунд)
```
Расширенная настройка

  * SSL-сертификаты: если используете Nginx, настройте пути к сертификатам в nginx.conf.
  * Резервное копирование: вся база данных находится в ./data/amnezia.db. Регулярно копируйте этот файл.
